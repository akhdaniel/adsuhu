# Добре дошли в Markdown Viewer

Това е **българската версия** на документацията.

## Функционалности

- ✅ Автоматично разпознаване на езика
- ✅ Връщане към английски ако няма превод
- ✅ Пълна поддръжка на Markdown
- ✅ Оцветяване на код

## Примерен код
python def zdravei(): print("Здравей свят!")

## Как работи

Системата разпознава езиковата настройка на Odoo и зарежда съответния файл:
- `en_US` → зарежда `readme.md`
- `bg_BG` → зарежда `readme.bg.md`
- Ако локализиран файл не съществува → връща се към `readme.md`
